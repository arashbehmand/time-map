Time-Map

Interactive tools for visualising travel time and finding fair meeting points, powered by Mapbox APIs, MapLibre GL JS and WebGL2.

Isochrone Warp Map

A map that bends to travel time. Isochrone boundaries become concentric circles — so equal travel time always appears at equal distance from you, regardless of direction.

Meetup Optimizer

Find the fairest meeting point for a group with different starting locations and transport modes. Minimises the worst-case (or weighted average) travel time using H3 hexagonal search.
